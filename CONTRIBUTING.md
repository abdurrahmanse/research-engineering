# Contributing

This is a personal learning repository. Conventions:

1. Each directory is self-contained with a `00_README.md`.
2. Files are numbered by learning order (`01_…`, `02_…`).
3. Code uses `snake_case`; docs use `kebab-case.md`.
4. Each module should be runnable and tested in isolation.
5. Refactor before adding new files; don't accumulate code smells.
