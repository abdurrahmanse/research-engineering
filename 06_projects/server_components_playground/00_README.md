# 07 — Server Components Playground

A hands-on playground for the React Server Components paradigm.
Server-first data fetching, client islands, and streaming.

## Files

- `01_server_data_fetcher.ts` — Async server component that fetches without waterfalls
- `02_client_island.tsx` — Minimal client component embedded in a server tree
- `03_streaming_suspense.tsx` — Suspense boundaries with streaming HTML chunks

## Stack

TypeScript · React 19 / Next.js App Router · RSC · Suspense
